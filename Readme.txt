This plugin allows you to control a SIY dust cover controller ( https://create.arduino.cc/projecthub/cfar/automated-telescope-dust-cover-controller-2226fc ) from TheSkyX Pro.
Make sure you use the rdc_fw_1_1servo_z6y6V9QJ5R.ino Arduino sketch.
Pre-compiled versions for OS X, Windows, Unbuntu X86_64 and Rapsberry Pi Pixel can be found there :
http://www.rti-zone.org/macosx_x2filterwheel_plugins.php
